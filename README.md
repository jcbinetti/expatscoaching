# expatscoaching — Anna Leonenko · Expat Coach

Zweisprachige (Russisch / Deutsch) Website für **Anna Leonenko** — Expat-Coach für berufliche Verwirklichung, Selbstständigkeit und Begleitung in Deutschland für russischsprachige Menschen.

**Live:** https://expatscoaching.de

## Design-System
- Zweisprachig über `.ru`/`.de`-Spans + `<html lang>`-Umschalter.
- Palette: Pfirsich (`#F0D3BC`), Himmelblau (`#AECEDD`), Grau-Blau-Akzent (`#6E9DB5`), Creme (`#FAF4EF`).
- Serifen-Überschriften (Georgia), eckige Kanten, Vollbreite-Bänder.

> Hinweis: Server-Backends (PHP) und interne Tools sind hier bewusst **nicht** enthalten. Dieses Repo ist die öffentliche Frontend-/Design-Quelle.
